# Samsung Global Product Sales Analysis

## Business Problem
This project analyzes global Samsung product sales data to identify revenue drivers, return patterns, and customer behavior across regions and sales channels.

## Tools Used
- PostgreSQL
- SQL (CTEs, Window Functions, Aggregations)
- Data Analysis

## Dataset
Samsung Global Product Sales Dataset (Kaggle)

## Key Analysis Performed
- Revenue by Sales Channel
- Return Rate by Product Category
- Discount Impact Analysis
- 5G vs Non-5G Performance
- Month-over-Month Revenue Growth

## Key Insights

### 1. Sales Channel Performance
Samsung Store and E-commerce Platform generated the highest revenue, indicating strong multi-channel distribution.

### 2. Return Analysis
Galaxy Z showed the highest return rate (~10.87%), suggesting potential product complexity or expectation mismatch.

### 4. 5G vs Non-5G
Non-5G devices currently contribute higher total revenue than 5G models.
